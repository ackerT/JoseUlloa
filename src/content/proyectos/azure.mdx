**Proyecto:**  
Este proyecto consistió en la creación y configuración de una infraestructura completa en Microsoft Azure mediante Terraform. Se implementaron diversas herramientas y servicios para automatizar el despliegue de recursos y asegurar una operación óptima y escalable para aplicaciones modernas.

<br /><br />

**Mi rol:**  

Fui responsable de diseñar y configurar componentes fundamentales de la infraestructura, incluyendo:
- **Servidor de Base de Datos:** Configuración y despliegue del servidor para alojar las bases de datos.
- **Bases de Datos:** Creación y gestión de las bases de datos necesarias para las aplicaciones.
- **Cache de Redis:** Implementación de Redis para optimizar el rendimiento mediante caching.
- **Function Apps para Mensajería Asíncrona:** Despliegue de Function Apps que facilitan la comunicación asíncrona entre servicios, mejorando la eficiencia operativa.

<br /><br />

**Resultados:**  

La infraestructura implementada permitió automatizar el despliegue de recursos en Azure, proporcionando un entorno seguro, escalable y de alto rendimiento. Esta solución facilitó la gestión de servicios críticos y optimizó la integración entre ellos, mejorando significativamente la eficiencia operativa y la capacidad de respuesta ante demandas variables.
